# RocketUAV-Sensor-Module
A breakout carrier plate for NAR L1 rockets. Fully populated, it integrates 2 barometers, an audio amplifier and an accelerometer sourced from Amazon, with a SparkFun Qwiic Adapter and an Adafruit Trinket M0. Pair with the I2C Service Module for a DIY Dual Deployment solution.
